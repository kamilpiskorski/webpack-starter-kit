# webpack-starter-kit

> Starter Kit based on [Webpack](https://webpack.js.org/) for work with ES6, TypeScript, Autoprefixer and SASS

## Setup

```bash
# install dependencies
npm install

# start auto reload server
npm start

# build for development
npm run dev

# build for production
npm run build
```

## Folder structure

    .
    ├── dist                # Compiled files
    └── src                 # Source files
         ├── assets
         │   ├── images
         │   └── ...
         ├── scripts
         │   └── app.js
         ├── styles
         │   └── app.scss
         └── index.html
